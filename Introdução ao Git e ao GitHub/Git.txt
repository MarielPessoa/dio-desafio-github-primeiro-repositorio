Oque é o Git?

O Git é um sistema de controle de versão distribuído, gratuito e de código aberto, usado para rastrear alterações em arquivos de projeto, especialmente em desenvolvimento de software. Ele permite que várias pessoas trabalhem juntas em um projeto, mantendo o histórico de alterações e facilitando a colaboração.
